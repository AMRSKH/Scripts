# бһ§бһ”бһҖбһҡбһҺбҹҚбһҹбһҳбҹ’бһўбһ¶бһҸбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ (Windows System Cleaners)

бһғбҹ’бһӣбһ¶бҹҶбһ„бһ•бҹ’бһ‘бһ»бһҖбһ“бҹҒбҹҮбһҳбһ¶бһ“бһ§бһ”бһҖбһҡбһҺбҹҚбһҹбһҳбҹ’бһўбһ¶бһҸбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ Windows бһ…бҹҶбһ“бһҪбһ“бһ–бһёбһҡ бһҠбҹӮбһӣбһ”бһ„бҹ’бһҖбһҫбһҸбһЎбһҫбһ„бһҠбҹ„бһҷ **AMSDev** бһҠбһҫбһҳбҹ’бһ”бһёбһҮбһҪбһҷбһұбҹ’бһҷбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҠбһҫбһҡбһӣбһҝбһ“ бһ“бһ·бһ„бһҳбһ¶бһ“бһ‘бҹҶбһ бҹҶбһ•бҹ’бһ‘бһ»бһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ‘бһјбһӣбһ¶бһҷбһҮбһ¶бһ„бһҳбһ»бһ“бҹ”
*(This repository contains two Windows system cleaner scripts developed by AMSDev to help keep your computer fast and save storage space.)*

---

## 1. AMS-SysCleaner.bat (бһҮбҹҶбһ“бһ¶бһ“бҹӢбһҗбҹ’бһҳбһё / Advanced Version)

бһ“бҹҒбҹҮбһӮбһәбһҮбһ¶бһ§бһ”бһҖбһҡбһҺбҹҚбһҹбһҳбҹ’бһўбһ¶бһҸбһҖбһҳбҹ’бһҡбһ·бһҸбһҒбҹ’бһ–бһҹбҹӢбһҠбҹӮбһӣбһҳбһ¶бһ“бһҳбҹүбһәбһ“бһ»бһҷбһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбҹ” бһңбһ¶бһ“бһ№бһ„бһ‘бһ¶бһҳбһ‘бһ¶бһҡбһҹбһ·бһ‘бҹ’бһ’бһ·бһҮбһ¶ Administrator бһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҠбһҫбһҳбҹ’бһ”бһёбһӣбһ»бһ”бһҜбһҖбһҹбһ¶бһҡбһҹбҹҠбһёбһҮбһҳбҹ’бһҡбҹ…бҹ”
*(This is an advanced cleaning tool with an interactive menu. It automatically requests Administrator privileges to clean deep system files.)*

**бһӣбһҖбҹ’бһҒбһҺбҹҲбһ–бһ·бһҹбҹҒбһҹбһҠбҹӮбһӣбһҳбһ¶бһ“ (Features):**
* **[1]** бһҹбһҳбҹ’бһўбһ¶бһҸбһҜбһҖбһҹбһ¶бһҡбһ”бһҺбҹ’бһҠбҹ„бҹҮбһўбһ¶бһҹбһ“бҹ’бһ“ (Clean Temporary Files: User, Windows, Prefetch)
* **[2]** бһҹбһҳбҹ’бһўбһ¶бһҸ Windows Update Cache (бһӣбһ»бһ”бһҜбһҖбһҹбһ¶бһҡ Update бһ…бһ¶бһҹбҹӢбҹ—бһҠбҹӮбһӣбһҹбҹҠбһёбһ‘бҹҶбһ бҹҶбһ•бҹ’бһ‘бһ»бһҖбһ’бҹҶ)
* **[3]** бһҹбһҳбҹ’бһўбһ¶бһҸбһ’бһ»бһ„бһҹбҹҶбһҡбһ¶бһҳ (Empty Recycle Bin)
* **[4]** бһӣбһ»бһ” DNS Cache (бһҮбҹҶбһ“бһҪбһҷбһҠбһӣбҹӢбһҖбһ¶бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶бһўбҹҠбһёбһ“бһ’бһәбһҺбһ·бһҸбһҠбһҫбһҡбһҷбһәбһҸ)
* **[5]** бһҳбһ»бһҒбһ„бһ¶бһҡ Deep Clean (бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҳбһ»бһҒбһ„бһ¶бһҡбһҹбһҳбҹ’бһўбһ¶бһҸбһҒбһ¶бһ„бһӣбһҫбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҖбҹ’бһ“бһ»бһ„бһ–бҹҒбһӣбһҸбҹӮбһҳбһҪбһҷ бһ бһҫбһҷ Restart Windows Explorer)

**бһҡбһ”бҹҖбһ”бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ:**
бһӮбҹ’бһҡбһ¶бһ“бҹӢбһҸбҹӮбһ…бһ»бһ…бһ–бһёбһҡбһҠбһ„ (Double-click) бһӣбһҫбһҜбһҖбһҹбһ¶бһҡ `AMS-SysCleaner.bat` бһ бһҫбһҷбһңбһ¶бһҷбһӣбҹҒбһҒ (бҹЎ-бҹҰ) бһҸбһ¶бһҳбһҳбҹүбһәбһ“бһ»бһҷбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ…бһ„бҹӢбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбҹ”

---

## 2. DelTemp.bat (бһҮбҹҶбһ“бһ¶бһ“бҹӢбһ…бһ¶бһҹбҹӢ / Classic Version)

бһ“бҹҒбҹҮбһӮбһәбһҮбһ¶бһ§бһ”бһҖбһҡбһҺбҹҚбһҹбһҳбҹ’бһўбһ¶бһҸбһҜбһҖбһҹбһ¶бһҡбһ”бһҺбҹ’бһҠбҹ„бҹҮбһўбһ¶бһҹбһ“бҹ’бһ“бһҮбҹҶбһ“бһ¶бһ“бҹӢбһҠбҹҶбһ”бһјбһ„ (Simple Script) бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһӣбһҝбһ“бһҡбһ бҹҗбһҹбһҠбҹ„бһҷбһҳбһ·бһ“бһ”бһ¶бһ…бҹӢбһҡбһҫбһҹбһҳбҹүбһәбһ“бһ»бһҷбҹ” бһңбһ¶бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бҹ”
*(This is the classic, simple temporary file cleaner. It runs automatically without a menu.)*

**бһўбҹ’бһңбһёбһҠбҹӮбһӣбһңбһ¶бһ’бҹ’бһңбһҫ (What it does):**
* бһ”бһ·бһ‘ Windows Explorer бһҮбһ¶бһ”бһҺбҹ’бһҠбҹ„бҹҮбһўбһ¶бһҹбһ“бҹ’бһ“бһҠбһҫбһҳбҹ’бһ”бһёбһҠбҹ„бҹҮбһҹбҹ„бһҜбһҖбһҹбһ¶бһҡ (Releases temp file locks)
* бһҹбһҳбҹ’бһўбһ¶бһҸ `%temp%`, `C:\Windows\Temp`, бһ“бһ·бһ„ `C:\Windows\Prefetch`
* бһ”бһҫбһҖ Windows Explorer бһЎбһҫбһ„бһңбһ·бһүбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ· (Restarts Explorer)

**бһҡбһ”бҹҖбһ”бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ:**
бһӮбҹ’бһҡбһ¶бһ“бҹӢбһҸбҹӮбһ…бһ»бһ…бһ–бһёбһҡбһҠбһ„ (Double-click) бһӣбһҫбһҜбһҖбһҹбһ¶бһҡ `DelTemp.bat` бһҠбһҫбһҳбҹ’бһ”бһёбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҖбһ¶бһҡбһҹбһҳбҹ’бһўбһ¶бһҸбһ—бҹ’бһӣбһ¶бһҳбҹ—бҹ”
